## カプセル化とは

フィールドを `private` にして、getter/setter 経由でのみアクセスさせることを「カプセル化」と言います。
こうすることで、不正な値が入ることを setter 内で防ぐことができます。

```php
// privateなので外から直接触れない
$user->age = -5;  // エラー！

// setter経由なら値を検証できる
$user->setAge(-5); // setter内でチェックして拒否できる
```
